# Moduler & Packager

A simple **menu-driven Python application** that demonstrates the use of various built-in Python modules such as **datetime, time, math, random, uuid**, and **file handling**. The project provides multiple utilities in one program and is ideal for beginners learning Python modules and functions.

---

## Features

### Datetime and Time Operations
- Display current date and time
- Calculate difference between two dates
- Format dates into different styles
- Stopwatch
- Countdown timer

### Mathematical Operations
- Calculate factorial
- Compound interest calculator
- Trigonometric calculations
- Area of circle and rectangle

### Random Data Generation
- Generate random numbers
- Generate random lists
- Create random passwords
- Generate 6-digit OTP

### UUID Generation
- Generate a unique UUID Version 4

### File Operations
- Create a text file
- Write data to a file
- Read data from a file
- Append data to a file

### Module Explorer
- View available attributes of Python modules using `dir()`
